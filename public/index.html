DashboardApp API

RESTfull API (version 1.0.0)

It is API for my project Dashboard App. This App have all CRUD functionality.

URL : http://domain:4000/api/v1/

You may see API's all source code at my GitHub page

DEVELOPED BY:

JSON WEB TOKEN

API's routs was protected by JWT.

After login you will get a token in response. To get a response from other routs you must add token to the header of your HTTP Request and refresh it every 30 minutes:


"Authorization" : "Bearer < token >"

API Routs:

.../api/v1
  • /login POST
    • POST/login/Autorize on the service
  • /logout POST
    • POST/login/Logout from the service
  • /refresh GET
    • GET /refresh/auth Authorization check and refresh token after reset App page
    • GET /refresh/token Refresh token after reset App page
  • /dashboard GET
    • GET /dashboard/{MAC} Get common device information by MAC Address
    • GET /dashboard/{MAC}/pingGet information about ping by MAC Address
    • GET /dashboard/{MAC}/dvbc Get information about DVB-C signal by MAC Address
  • /profile
    GETPATCHDELETE
    • GET /profile/ Get profile info
    • PATCH /profile/ Change profile info without password
    • PATCH /profile/passwordChange profile password
    • DELETE /profile/ Delete profile page from the service
  • /users
    GETPOSTPATCHDELETE
    • GET /users/ Get all users
    • GET /users/{userId} Get profile by userId
    • POST /users/ Add new user
    • PATCH /users/{userId} Change profile by userId without password
    • PATCH /users/password/{userId}Change user password
    • DELETE /users/ Delete user from the service
